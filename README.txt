# HTML LIVE VIEWER — REALTIME PREVIEW (HTML)

Aplikasi **HTML Live Viewer / Playground** berbasis HTML, CSS, dan JavaScript (Vanilla) yang berjalan sepenuhnya di browser. Dirancang untuk menulis, mengedit, dan melihat hasil HTML secara real-time tanpa instalasi.

## 🚀 Fitur Utama

### ✏️ Live Editor & Preview
- Split screen: editor (kiri) + preview (kanan)  
- Hasil HTML diperbarui **real-time** di iframe  
- Debounce input untuk performa lebih halus  

### 🔧 UI & Interaksi
- **Resizable divider** (drag untuk atur lebar panel)  
- **Fullscreen Editor**  
- **Fullscreen Preview**  
- **Mini Floating Editor** (draggable + resizable)  
- Sinkronisasi otomatis antara editor utama dan mini editor  

### 🎨 Tampilan
- Tema dark modern  
- Monospaced font untuk pengalaman coding nyaman  
- Layout responsif (stack di layar kecil)

---

## 🛠 Teknologi yang Digunakan
- **HTML** → Struktur UI dan layout  
- **CSS (Vanilla)** → Tema dark, split view, dan floating panel  
- **JavaScript (Vanilla)** → Live preview, fullscreen, drag, dan mini editor  
- **iframe (srcdoc)** → Rendering HTML secara aman di browser  
- **Google Fonts (OFL):**  
  - Inter  
  - JetBrains Mono
